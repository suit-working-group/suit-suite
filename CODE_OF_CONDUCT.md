# Code of Conduct

The SUIT working group adopts the **Contributor Covenant, version 2.1**, as its code of conduct.

The full text is maintained by the Contributor Covenant project and is available here:

- <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Our pledge (summary)

We pledge to make participation in the SUIT community a harassment-free experience for everyone, regardless of background or identity, and to act in ways that contribute to an open, welcoming, diverse, and healthy community.

## Scope

This Code of Conduct applies within all project spaces — issues, pull requests, discussions, and any communication channel used by the working group — and when an individual is representing the project in public spaces.

## Reporting

To report unacceptable behaviour, open a **confidential** matter with the maintainer. Because this repository is ticket-centralized, sensitive reports should not be filed as public issues: contact the maintainer directly — by email at [nicolas.guelfi@uni.lu](mailto:nicolas.guelfi@uni.lu), or through their GitHub profile (**@nicolasguelfi**). All reports will be reviewed and investigated promptly and fairly.

Enforcement follows the Contributor Covenant's **Enforcement Guidelines** (Correction → Warning → Temporary Ban → Permanent Ban), as described in the linked full text.
