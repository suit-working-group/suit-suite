# Evidence Methodology

## Five-Layer Evidence Architecture
When building an argument or defending a thesis, construct evidence at five independent levels:

### Layer 1 — Binding Legal Instruments
- EU primary law: Charter of Fundamental Rights, ECHR
- EU secondary legislation: Regulations (GDPR, AI Act), Directives (NIS2, 2000/78/EC)
- Case law: CJEU judgments (Schrems, DRI, Scarlet Extended), ECtHR judgments (Barbulescu, Sorguc)
- National transposition: local implementing laws
- Always cite specific articles, clauses, or paragraphs

### Layer 2 — Normative Frameworks and Standards
- Technical standards: NIST SP series, ISO/IEC 27001, ISO/IEC 27002
- Governance frameworks: COBIT 2019, ITIL
- Cybersecurity frameworks: NIST CSF 2.0, Zero Trust (NIST SP 800-207)
- Reference specific controls, functions, or processes (e.g., "NIST CSF 2.0 GV function")

### Layer 3 — Sector Consensus
- Professional body publications: EDUCAUSE, GÉANT, JISC, REN-ISAC
- Survey data: EDUCAUSE Top 10, Verizon DBIR, Sophos sector reports
- Sector guidelines: EDPB, ENISA, CNIL recommendations
- Note the representative sample size and methodology when citing surveys

### Layer 4 — Empirical Data
- Quantified performance measurements (throughput, latency, incident metrics)
- Incident analysis with root cause identification
- Before/after comparisons from documented implementations
- Statistical significance or sample size when available
- Always include units, time periods, and measurement context

### Layer 5 — Institutional Precedents
- Named universities with verifiable, documented practices
- Prefer institutions with publicly available documentation
- Include specific organizational details (team names, governance structures, policy documents)
- Minimum 5 diverse institutions across different countries/systems
- Verify each precedent claim against the institution's own published documentation

## Convergence Principle
- A thesis is considered robust when supported by at least 3 independent layers
- If any single layer is successfully challenged, the remaining layers must sustain the argument
- When only 1-2 layers are available, explicitly acknowledge the limitation
- Present the strongest layer first, then corroborate with others

## Counter-Evidence Handling
- When evidence exists that could weaken a thesis, cite it explicitly
- Explain why the counter-evidence does not invalidate the thesis (scope, context, methodology)
- Never suppress inconvenient evidence — acknowledge and contextualize it
